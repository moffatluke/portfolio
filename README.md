# Portfolio

Personal portfolio site built with React and Tailwind CSS. Features smooth animations, a projects showcase, skills section, and a contact form.

## Tech Stack

- **React 19** — component-based UI
- **Vite 7** — build tool and dev server
- **Tailwind CSS 4** — utility-first styling
- **Framer Motion** — page and element animations
- **shadcn/ui** — accessible UI primitives

## Sections

- **Hero** — intro with animated background
- **About** — background and interests
- **Skills** — tech stack with visual tags
- **Projects** — selected work with descriptions and tech tags
- **Contact** — contact form

## Running Locally

```bash
git clone https://github.com/moffatluke/portfolio.git
cd portfolio
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.
